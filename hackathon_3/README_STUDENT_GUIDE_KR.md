# 🧪 프로젝트 가이드
## micro:bit로 만드는 실험실 실시간 모니터링 시스템

---

## 1️⃣ 프로젝트 소개

### 우리가 만들 것
우리는 **micro:bit와 센서**를 사용해서
실험실의 상태를 **실시간으로 측정하고**,
컴퓨터 화면에서 **그래프로 확인하는 시스템**을 만듭니다.

측정 예:
- 🌡 온도
- 💧 습도
- 🫁 공기 상태(가스)

---

## 2️⃣ 전체 흐름 이해하기

```
센서 → micro:bit → 컴퓨터 → 웹 화면(그래프)
```

- 센서: 주변 환경을 숫자로 바꾼다
- micro:bit: 숫자를 컴퓨터로 보낸다
- 컴퓨터: 숫자를 모아서 보여준다

---

## 3️⃣ 준비물 확인

### 한 팀당 준비물
- micro:bit v2 (1개)
- Sensor Shield v2.0 (1개)
- USB 케이블 (1개)
- 센서 (온도 / 습도 / 가스 중 1개 이상)
- 실험실 컴퓨터 또는 라즈베리파이

👉 시작 전에 **부품 이름을 정확히 확인**하세요.

---

## 4️⃣ 센서 연결하기 (손으로 직접!)

### 기본 연결 규칙 (매우 중요)
- VCC → 3V
- GND → GND
- SIGNAL → P0 (또는 P1)

### 왜 GND가 필요할까?
전기가 **돌아오는 길**이 없으면 센서는 동작하지 않습니다.

📝 활동
- 연결 후 친구에게 설명해 보세요:
  > “이 선은 왜 여기에 연결했을까?”

## 5️⃣ micro:bit 코딩하기

### 목표
> “1초마다 센서 값을 컴퓨터로 보내기”

### MicroPython 코드 예제
```python
from microbit import *
import utime

while True:
    value = temperature()
    timestamp = utime.ticks_ms()
    print(f"{timestamp},{value}")
    sleep(1000)
```

### 코드 이해하기
- `while True` : 계속 반복
- `temperature()` : 센서 값 읽기
- `print()` : 컴퓨터로 데이터 보내기

---

## 6️⃣ 데이터가 잘 오는지 확인하기

### 확인 방법
1. micro:bit를 USB로 컴퓨터에 연결
2. 터미널(또는 Mu 에디터) 열기
3. 숫자가 계속 출력되는지 확인

예시:
```
123456,24
124456,24
125456,25
```
